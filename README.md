# Hidden_Markov_Model_Strategy_Tester
Regime-Based Trading App uses a 7-component Hidden Markov Model to detect Bull/Bear/Sideways market regimes in crypto. Combines HMM with an 8-point confirmation system (RSI, MACD, ADX, EMAs, etc.) for trade entries. Features trailing stops, multiple risk modes, and fully customizable parameters via Streamlit dashboard.
